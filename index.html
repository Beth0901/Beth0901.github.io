Evaluation of project

This section will discuss the positives, in the left-hand columns, and the negatives, in the right-hand columns, of this project.

Skills learnt

This project allowed for numerous skills to be learnt across the various technologies discussed. These skills are valuable knowledge to have for creating similar projects in the future.

Difference in final design

A downside to this project was that the final design achieved did not meet the intended design specification set out by the team. This means that it did not meet expectations with the final product.

Finished product

One positive of the project is that there was a final artefact to show. Although it does not function as intended nor meet the original designs, each component is still physically made.

Lack of articulation

The end product of the project did not have articulation as it was intended to. This lack of articulation is a major downside to the project and means it did not meet the necessary requirements for the project.
